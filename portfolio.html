Design Portfolio

Creating a home for my project work

Problem

My design portfolio is an online platform that displays my work, design skills, and personality to other designers, potential recruiters, and hiring managers.

It's also a medium for me to practice and demonstrate front-end development and web design.

TIMELINE
Nov 2021
Present

HTML

I began my website by writing HTML documentation using the IDE, Atom. To ensure accessibility, I used a mobile-first approach. To focus on content over style, I referenced sample wireframes.



Sample Wireframes Structure

CSS

As I was writing CSS to emulate the sample wireframes, I noticed I was sinking too much time in constantly tweaking the style. So, I decided to create wireframes to guide my design.

Seeing my design from the development perspective revealed the importance of creating clear and consistent assets as a designer.

JavaScript

I created a JavaScript file and added jQuery to my HTML documents to add advanced interactivity to my site, like a menu button for mobile users.

I also explored CSS interactions and animations to add more visual flair to my site. I kept these fairly simple, as it was important that they actually conveyed meaning and weren't distracting.

Iteration

I conducted five usability tests to assess my site's usability. Participants had no issue with the test scenarios, but all commented on my site being too plain.

To address this, I added dynamic page colors and parallax images to add fluidity to my portfolio. These additions were well received when critiquing my site with other designers.

Lessons

  1. I learned that efficient development starts with a solid design, even if an assignment doesn't ask you for one.
  2. Since this project was my first attempt at developing a website, I hand coded most of it for practice. I'd likely use frameworks in the future to say the least.
  3. I initially used Atom per assignment instructions but eventually switched to VS Code as I was more familiar with it. This helped me focus on learning one thing at a time.
